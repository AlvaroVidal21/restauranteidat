# Mi Diagrama de Base de Datos

Este proyecto contiene un esquema de base de datos definido en formato DBML y tipos de TypeScript relacionados. A continuación se describen los archivos principales del proyecto:

## Estructura del Proyecto

```
mi-diagrama-bd
├── src
│   ├── schema.dbml        # Esquema de la base de datos en formato DBML
│   └── types
│       └── index.ts       # Tipos e interfaces para TypeScript
├── package.json            # Configuración de npm
├── tsconfig.json           # Configuración de TypeScript
└── README.md               # Documentación del proyecto
```

## Descripción de Archivos

- **src/schema.dbml**: Este archivo contiene el esquema de la base de datos, donde se definen las tablas, relaciones y campos.

- **src/types/index.ts**: Este archivo exporta tipos e interfaces que se utilizan para definir la estructura de datos en TypeScript, relacionados con el esquema de la base de datos.

- **tsconfig.json**: Archivo de configuración para TypeScript que especifica las opciones del compilador y los archivos a incluir en la compilación.

- **package.json**: Archivo de configuración para npm que enumera las dependencias y scripts necesarios para el proyecto.

## Instalación

Para instalar las dependencias del proyecto, ejecute el siguiente comando:

```
npm install
```

## Compilación

Para compilar el proyecto, use el siguiente comando:

```
npm run build
```
